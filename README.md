# Pomodoro 🍅

A simple Pomodoro Timer.

![Demo](demo.png)

https://wenfangdu.github.io/pomodoro/

Timer shows in the page title.  
Press <kbd>Space</kbd> to pause/unpause, refresh the page to reset the timer.  
Optimized for performance, works offline.
